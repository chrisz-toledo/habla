# Changelog

All notable changes to Habla will be documented in this file.

## [0.1.0] - 2026-04-11

### Added
- Initial release of Habla DSL
- Lexer with full INDENT/DEDENT support
- Recursive descent parser
- Three compilation backends: Python, C, Rust
- Cybersecurity modules: scanner, recon, capture, attack, analysis, report
- CLI: `habla run`, `habla compile --target [python|c|rust]`, `habla repl`
- ASCII normalizer for Spanish diacritics
- 7 example programs
- Full test suite
- Documentation: spec, tutorial, cybersec cookbook, LLM guide, design decisions
- Standard library stubs: red, archivo, texto, crypto
